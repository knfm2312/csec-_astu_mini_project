# Yabsira Abebe Portfolio Website

This is my personal portfolio website built using HTML CSS and JavaScript.  
It showcases my profile background skills and projects with a responsive design.

---

## Features

- About Me section with profile photo  
- Skills section listing technical abilities  
- Projects section with descriptions of completed work  
- Contact form for sending messages  
- Responsive design for mobile and desktop

---

## Technologies Used

- HTML5  
- CSS3  
- JavaScript
